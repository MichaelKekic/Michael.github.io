Vending Machine Software Capstone




Technologies Used


Java Eclipse BitBucket
OOP Fundamentals FileInput/Output Command Line Interface

Our task was to complete a Command Line Interface(CLI) program for a Vending Machine. The program was built in Eclipse with Java using object oriented principles as well as FileInput/Output. The vending machine software had to be able to read a text file containing the vending machine items, locations, and price.

The user then had to be able to enter the (CLI) and choose to see available items, insert money, and exit the program. The CLI would display the total money entered, and the available items to choose. If the item was sold out they would have a prompt telling them so. Meanwhile the program writes a log of user inputs, and transactions. There is also a separate log created to keep track of all total sales.

Campground Reservation Software Capstone




Technologies Used


Java Eclipse BitBucket
OOP Fundamentals Command Line Interface PostgresSQL
DbVisualizer Command Line Interface JDBC's & DAO's

Our task was to complete a Command Line Interface(CLI) program for a National Parks Service allowing users to book reservations to selected campsites. The program was built in Eclipse with Java using object oriented principles as well as SQL statements being written using DbVisualizer and PostgresSQL. The National Park Service software pull in data from our Postgres database to show users the parks description, amenities, and availability.

The user then had to be able to enter the (CLI) and choose to see park descriptions, prices, available dates, and exit the program. The CLI would display a list of National Parks, and the descriptions. The user could then choose to view other parks or make a reservation for specific dates. Dates that were not for specific campsites would not be available options. The program would post their reservation to the database and ask if they wanted to make another or exit the program.
